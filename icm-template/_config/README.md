# Configuration

Store project-wide configuration here. These files are configured once during setup and referenced by every stage.

## Suggested Files
- `brand-voice.md` — Tone, style, vocabulary guidelines
- `design-system.md` — Colors, typography, layout rules
- `audience.md` — Target audience profiles and preferences
- `constraints.md` — Budget limits, word counts, format requirements

## Rules
- Configure once, reference everywhere
- Stages point to these files — they never copy the content
- Update here, and all stages pick up the change automatically
